What is Seamless for OpenSCAD?

The Seamless compiler can import OpenSCAD modules directly from a online module library. It works as a precompiler.

Example: If you want to design a wall mount for your arduino, you may need to measure the positions of the mounting holes of your arduino. It's far easier if you search for a module that can create those holes for you, and just call ~px-arduino_mega_with_sainsmart_lcd-v1(only_make_bores = true);

How can i use it?

Download the seamless compiler for Windows or Linux and install it. Start it.

The compiler (also known as client) simply watches a whole directory including subfolders for changes. Whenever you edit or create a .scad file, the compiler instantly compiles it into a .compiled.scad file.

Choose a directory and click "Start"

Depending on your habbits, you may have to adapt a new habbit: You can not use the built-in OpenSCAD text editor anymore.

You have to use an external editor. I recommend Notepad++ for Windows and gedit or kate for Linux. Create and edit your my_new_design.scad file in your favorite editor.

Open OpenSCAD. Load the my_new_design.compiled.scad file into OpenSCAD. You can hide the built-in editor in OpenSCAD via Menu->View->Hide editor. You also need to enable Menu->Design->'Automatic reload and compile'.

Now you can edit your design. Whenever you save your file (Ctrl + S) it precompiles (searches for online modules) and compiles automatically.
